Intro

Micro is simple 4KB client side library. It has no dependencies, so plain javascript that works across all browsers. Main purpose is to turn any "hard-reload" website to Ajax driven SPA. It fit's fine with any client side libraries like bootstrap, materialize etc. But with dynamic sites too. It's environment agnostic and very suitable to "ajaxify"" existing projects.

Micro is combined from Router and Tpl modules.

Router is a client side module based on history api. That means you can use "pretty" links with your application by writing url's like http://www.example.com/page/subpage.

Tpl module has features like

  • - Loading templates
  • - Basic data tpl data forwarding
  • - Enter animations
We will see in the last section and on examples page how you can easily combine templates with routes to created smoth and user friendly single page website.

Installation

    $ npm install micro-spa

or by embedding directly on page

    <script src="https://unpkg.com/micro-spa@latest"></script>

you can also clone https://github.com/sgoran/micro.git to try out examples

How to use it?

You need two things

1. instruct your server to redirect all non-existing file and folder request to your index file

Apache example 

        
        RewriteEngine On
        RewriteBase /
        RewriteRule ^index\.php$ - [L]
        RewriteCond %{REQUEST_FILENAME} !-f
        RewriteCond %{REQUEST_FILENAME} !-d
        RewriteRule . /index.php [L]

Ngnix example 

        location / {
            try_files   $uri $uri/ /index.php?$args;
        }

Express example

You will probably write your own logic with few lines or user some middleware. You can check my server here

2. Create Micro object with few properties

Simples usage is

        new Micro({
            config: {
                container: '#app',
                tplDir: '/tpl'
            },
            router: [{
                match: '/',
                title: 'Home Page',
                src: "home.html"
            },{
                match: '/example',
                title: 'Example Page',
                src: "examples.html"
            }]
        });
