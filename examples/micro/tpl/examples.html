Creating Micro instance

Example with one page and one route 

      var micro = new Micro({
            config: {
                container: '#app',
                tplDir: '/tpl'
            },
            router: [{
                match: '/',
                title: 'Home Page',
                src: "home.html"
            }]
        });

This code will load html from home.html file which is in tpl folder.
