Intro

Micro is very simple 4KB client side single page application helper. It has no dependencies, so plain javascript that works across all browsers. Main purpose is to turn any "hard-reload" website to Ajax driven SPA. It fit's very fine with any client side libraries like bootstrap, materialize etc. But with dynamic sites too. It's environment agnostic and very suitable to ajaxify existing projects.

Micro is combined from Router and Tpl modules.

Router is a client side module based on history api. That means you can use "pretty" links with your application by writing url's like http://www.example.com/page/subpage.

Tpl module has features like

  • - Loading templates
  • - Basic data tpl data forwarding
  • - Enter animations
We will see in the last section and on examples page how you can easily combine templates with routes to created smoth and user friendly single page website.
